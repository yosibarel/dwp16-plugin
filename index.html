Welcome to DWP-16

The ultimate audio plugin for music production.

Why Choose DWP-16?

DWP-16 is a 16-channel sampler that offers advanced sample manipulations like pitch shifting, reverse, loop, and FM synthesis.

Import Custom Audio

Import and edit samples directly within the plugin. Easily manipulate audio with pitch, reverse, loop, and more.

Built-in Effects

Each channel features effects like delay, filter, reverb, flanger, chorus, phaser, and distortion.

Advanced Modulation

Use LFOs, Curve Sequencer, and Random Note features to create dynamic soundscapes.
